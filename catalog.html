Pangeo Data Catalog¶

These datasets are directly accessible from the Google Cloud Pangeo deployment: pangeo.pydata.org.

  • newmann_netcdf

  • Newman met ensemble in netcdf

  • Format: netcdf
  • Container: xarray

  • Location: /gcs/newmann-met-ensemble-netcdf/conus_ens_00*.nc
  • import intake
catalog_url = 'https://github.com/pangeo-data/pangeo/raw/master/gce/catalog.yaml'
newmann_netcdf = intake.Catalog(catalog_url).newmann_netcdf.to_dask()
